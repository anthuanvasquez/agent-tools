# Example Skill

This is a template to demonstrate the folder-based skill structure.

## When to use this skill
- Use this as a reference when creating new skills.
- Use this to verify the `skills` tool functionality.

## How to use it
1. Create a folder in `.agent/skills/` with the skill name (kebab-case).
2. Create a `SKILL.md` file inside that folder.
3. Add YAML frontmatter with `name` and `description`.
4. Write detailed instructions in Markdown.
